# F1 History — Claude Instructions

This is an application to rewatch races and immerse yourself in the past. Users want to use this app to get all information for a race before rewatching. Spoilers for a race should never be shown without an additional click and most information - such as stats - should be of before the race. Any information is always fixed to the race-weekend.

## Code style

- TypeScript/Astro files are linted with ESLint and type-checked with `pnpm check` on session stop. Run `pnpm lint --fix && pnpm check` manually if you want to clean up mid-session.
